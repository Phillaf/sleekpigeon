# SleekPigeons

Explore financial and economic data.

## Install

```
git clone https://github.com/phillaf/sleekpigeons.git && cd sleekpigeons
echo "FINNHUB_KEY=XXXXXXXXXXXXXX" > .env
echo "QUANDL_KEY=XXXXXXXXXXXXXX" >> .env
docker-compose up -d
curl 127.0.0.1/news
```
